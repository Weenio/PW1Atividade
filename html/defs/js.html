Java Script

O Básico

Java Script é uma linguagem de programação interpretada estruturada, de script em alto nível. Junto com HTML5 e CSS3, o JavaScript é uma das três principais tecnologias da Rede Mundial de Computadores. Apesar do nome, JavaScript nada tem a ver com a linguagem de programação "Java".

Por oferecer a possibilidade de interações da página com o usuário e criação de conteúdos dinâmicos, o JS (abreviação de "Java Script"), é amplamente usado na grande maioria das páginas de hoje em dia, sendo quase impossível encontrar uma página que não use JavaScript para uma única interação se quer.

O JavaScript tem uma série de vantagens que fazem com ele seja a melhor escolha entre seus competidores. Dentre elas estão: Não precisar compilar o código, é uma linguagem fácil de aprender e é mais rápido e mais leve que outras linguagens que podem ser ultilizadas para programação web.

Todas as linguagens de programação tem as suas fraquezas, oque o deixa também vulnerável à brechas na segurança, podendo executar códigos maliciosos no dispositivo do usuário. Para o JavaScript é principalmente a sua popularidade, oque atrai os famosos hackers. Dentre as outras fraquezas estão: ele nem sempre tem suporte para todos os navegadores ou dispositivos, pode ser executado de diferentes formas em diferentes dispositivos e os seus códigos costumam ser bem grandes.

O principal do Java Script: functions

Ao declarar um botão em um documento HTML, é recomendado que se declare a função que ele vai acabar executando na sua linha de código. Java Script é uma linguagem também orientada por um objeto, isso é, uma função. O botão de trocar o tema escuro para claro nesse site, usa essa mesma lógica. Com essas functions é que a mágica do JavaScript acontece. Com elas, é possível manipular elementos do documento, interagir com APIs, validar formulários, criar animações complexas e enviar e receber dados do servidor.

O JavaScript desempenha um papel fundamental no desenvolvimento web, permitindo criar páginas e aplicativos interativos, dinâmicos e responsivos, tanto no navegador quanto no servidor.

No geral, as inteações do tipo "functions" recebem sempre esse mesmo processo e aparência: Primeiro você declara um nome para a função, depois você coloca o código a ser executado entre os colchetes, e encerra fechando o código.
