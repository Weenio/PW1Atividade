HTML: Linguagem de Marcação de Hipertexto

O Conceito básico

A Linguagem de Marcação de Hipertexto é a principal linguagem ultilizada na front-end de sites, para mostrar os mais diversos conteúdos ao visitante do mesmo. Essa linguagem de marcação conta com tags para mostrar texto de título, parágrafos, imagens, inserção de scripts e de estilos em CSS3. Como HTML5 é uma linguagem muito básica, ela é mais usada com outras linguagens para descrever o estilo da página, como o anteriormente citado CSS3, e para inserir funcionalidades, no JS.

O "Hipertexto" ao qual a nomenclatura se refere, se dá aos links que podem ser feitos entre os sites, sejam eles complementares entre o próprio site (como o caso dessa atividade), ou entre sites diferentes (como por exemplo um site que leva à um site de um anunciante). A capacidade dos links é um fator muito importante para internet, uma vez que carregar páginas de terceiros possa ajudar à espalhar o conhecimento dos mesmos. Ao compartilhar conteúdo na internet e vincular esses conteúdos em sites de terceiros, você se torna automaticamente um membro ativo da WWW (World Wide Web, ou Rede Mundial de Computadores).

Tags: o principal do HTML

A marcação do HTML é tudo oque se vê na imagem a esquerda. São essas tags que fazem tudo isso ser mostrado na tela para o usuário. Claro que, no caso do exemplo, está tudo estilizado, por isso essas tags são maiores do que realmente seriam se fosse de fato, um HTML puro.

Essas tags, também chamadas de elementos, são separadas do resto do código pelos caracteres "<" e ">" . Como você deve ter notado na imagem, a tag , mas fecha (com "/"). A marcação de de "fechar" o elemento é muito importante, pois ela delimita o espaço que a tag vai impactar na página. Sem ela, varias classes em css impactariam a página ao mesmo tempo, fazendo com que a página puldesse se tornar inaproveitável.

Existe uma quantidade gigantesca tags para serem usadas no HTML, ainda mais pelo fato de terem outras 4 versões da linguagem de formatação (oque implica que muitas das tags foram herdadas ou adaptadas de versões antigas, hoje datadas). Mas, dentre todos esses elementos, pode-se destacar a tag "body", que não é a mais importante só porque existe a "!DOCTYPE html". A tag delimita o conteúdo principal do documento que será exibido no navegador da web. Todas as informações visíveis ao usuário, como texto, imagens, links e elementos interativos, devem ser inseridas dentro dessa tag. Dentro do código, a tag body é a penultima a ser fechada, sendo a ultima a tag "html". A grosso modo, a tag body é responsável por colocar na tela todo o conteúdo a ser exibido, e tudo oque é inserido no documento que não esteja dentro das delimitações dessa tag é automaticamente excluido da exibição, oque nos leva há uma parte que está presente na maioria dos documentos HTML: A tag "head".

O cabeçalho

A tag head é uma das principais tags utilizadas na estruturação de uma página HTML. Ela fornece informações sobre o documento, mas não exibe conteúdo visível ao usuário. Essa tag contém metadados, referências a arquivos externos, scripts, estilos e outras informações importantes relacionadas ao documento HTML.

Dentro da tag head, pode-se incluir várias subtags, como a "title", "meta", "link", "script" e "style". Cada tag dessas tem uma função em específico dentro da head, e são elas:

• "title": Define o título da página, que geralmente é exibido na barra de título do navegador ou na guia. É um elemento obrigatório dentro da tag "head".
• "meta": Fornece os metadados sobre o documento HTML, como a descrição da página, palavras-chave relevantes, autor, codificação de caracteres e etc. Os metadados ajudam os motores de busca a entenderem melhor o conteúdo da página.
• "link": Usado para vincular um arquivo externo à página HTML, como um arquivo CSS, um ícone de página, uma fonte personalizada ou um script de comando em JavaScript.
• "script": Permite a inclusão de scripts JavaScript na página. Esses scripts podem ser usados para adicionar funcionalidades interativas à página, manipular elementos da página, enviar ou receber dados do servidor, entre outras coisas.
• "style": Utilizado para definir estilos CSS internos que serão aplicados à página HTML. Os estilos definidos dentro da tag "style" afetam apenas o documento atual.

A tag "head" é sempre posicionada antes da tag "body" dentro do HTML e serve como uma caixa para todas as informações e recursos para que a página funcine e para que a página seja formatada adequadamente. Embora seu conteúdo não seja visível diretamente ao usuário, ele desempenha um papel crucial na estruturação e na apresentação da página.
