CSS: Cascading Style Sheets

O Básico

A Folha de Estilos em Cascatas é uma linguagem de estilização que serve para dar visuais para uma página HTML. A função principal do CSS é descrever como os elementos vão aparecer na tela e/ou em outras mídias.

CSS é uma das linguagens principai da open web, e é padronizada de a cordo com as normas da W3C. Assim como HTML, temos versões datadas do CSS também, e do mesmo modo, algumas características foram heradas/aprimoradas das versões anteriores. A versão CSS1 está atualmente datada, e o CSS2.1 é uma recomendação ao atual CSS3, que dividido em pequenos módulos está mostrando avanços significativos em suas padronizações.

A diferença entre um site que ultiliza CSS e um que não usa é gigantesca. Sites que não usam esse artifício normalmente tem o texto preto e/ou azul e o fundo branco. Nos primórdios da internet, não era difil que as páginas fossem apresentadas dessa forma, seja por mal carregamento da folha de estilo (pela fraca internet da época) ou pelo fato do criador da página ter optado por não colocar o CSS.

Aparência de um codigo em CSS3

Como antes citado, o css tem o objetivo de estilizar a página. Um código em css tem várias definições que podem ter efeitos estéticos na página, e eles vão desde o tamanho da fonte, até animações básicas. A propriedade mais comum ao usar um css é a margem, com ela, podemos delimitar uma margem para a página mostrar o texto, sem ela ter que ficar colada nas bordas da tela. Pode-se também modificar tags já existentes (como é o caso desse exemplo), para que eles tomem propriedades específicas. Se as instruções forem colocadas no css, todas as tags modificadas na página tomaram os parâmetros. É possivel determinar um estilo específico para uma tag específica na página, sem que as demais tomem esse estilo, e sem a necessidade de fazer vários estilos simples no css com a funcionalidade chamada "style". Nesta página, usamos o "style" nas imagens para definir se a orientação delas será na esquerda da página, ou na direita. Sem essa funcionalidade, teriamos que colocar dois estilos específicos no css para cada orientação de imagem.

Estilos direto na tag do html: funcionalidade "Style:"

Como dito antes, a função "style" é uma propriedade que pode ser aplicada a várias tags HTML para definir estilos e aparência visuais de tags específicas, sem a necessidade de usar uma folha de estilos CSS externa. A propriedade "style" permite que você defina estilos em linha, tornando-o útil para aplicar estilos específicos a elementos específicos sem afetar outros elementos da página, como é o caso da imagem de exemplo.

Ao usar a função "style", você pode definir literalmente qualquer propriedade nativa do CSS3. Quando são essas propriedades são mais que uma, elas são escritas com pares de colchetes, separadas por ponto e vírgula (;).

Essa propriedade é uma ferramenta muito forte, porém, se usada de forma exessiva, pode causar uma certa dificuldade na hora de arrumar possíveis erros, ou na hora de atualizar o código por algum motivo. O aconcelhado é colocar essas especificações dentro de alguma folha de estilos, ao invés de descreve-las na hora, quando for colocar uma nova tag no documento. As folhas de estilo oferecem também uma extrema flexbilidade na ultilização, já que os estilos são globais e podem ser reutilizados por outras tags apenas declarando um outra funcionalidade chamada "class". Mas como neste caso, só precisariamos fazer alguns ajustes básicos, é muito mais conpensatório usar o "style", ao invés de declarar um código de estilização apenas para mudar a imagem de lugar.

Bootstrap

O Bootstrap é um dos frameworks front-end mais populares para desenvolvimento web. Ele foi criado por Mark Otto e Jacó Thornton e é uma bibliotecas de código aberto que contém estilos CSS pré-definidos e componentes interativos em scripts JavaScript para ajudar a criar interfaces responsivas e modernas de forma rápida e fácil.

O objetivo principal do Bootstrap é oferecer um ponto de partida sólido para o desenvolvimento de sites e aplicativos web, além de ajudar também iniciantes do front-end. Ele fornece uma série de componentes prontos para uso que ajudam a estruturar o layout e estilizar elementos, melhoramdo a experiência do usuário e facilitando a do desenvolvedor.

No geral, Bootstrap é abraçado pela comunidade de desenvolvedores front-end, por conta da sua praticidade. Além de tudo isso, é possível usar os estilos predefinidos do Bootstrap para entender ainda mais sobre CSS, e fazer os seus próprios estilos com esse aprendizado.
